PERCEIVABLE INTERFACE

An Interactive Dance Installation

  • Using Kinect sensor, Perceivable Interface transforms gestural data into symbolic, and creates an interactive live dance performance.  We take pleasure from seeing the relationship between the human form and its surroundings. This work targets this fascination and plays with the idea of the “virtual-self.”
