Making Open Data Dashboards More Relevant for Educational Data in Tanzania

  • World Bank has created a set of prototype open data dashboards for education to gather initial user feedback. My task as a participant of International Development Hackathon was to visualize these data in a more relevant and readable way for Tanzanians. I’ve created an iconography and a design theme in order to make sections of these dashboards more visually intuitive.
